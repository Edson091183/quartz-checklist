Quartz Checklist Studio (v3)

- Cada aba do Excel (17/12/2025) vira um checklist no menu lateral.
- Você pode editar texto, ordem, nº do modelo, observação, nome do checklist, e EXCLUIR LINHAS.
- Tudo salva localmente no seu navegador (localStorage).

PDF:
- Botão "Gerar PDF" usa jsPDF (offline) e inclui logo + rodapé institucional.
- Se o navegador bloquear downloads, use Ctrl+P (imprimir) na versão que abrir.

Dica:
- Se quiser voltar à base original do Excel, use "Recarregar do Excel (base)".
